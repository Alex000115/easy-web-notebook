# Easy Web Notebook

This is a very simple project for beginners.

You can use this repository to:
- Write daily notes
- Track learning progress
- Practice editing files
- Build a habit of daily GitHub commits

How to use:
1. Open any file
2. Add a small note
3. Save the file
4. Commit your update

Goal:
Learn a little every day.

Notes:
Write your daily notes below.
